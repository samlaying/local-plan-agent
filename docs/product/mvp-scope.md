# MVP Scope

## In Scope

- Natural-language request parsing.
- Mock POI, restaurant, activity and queue data.
- Plan generation with 1 main plan and 2 alternatives.
- Restaurant fit explanation for child-friendly and diet-friendly needs.
- Execution action list generation.
- Real map navigation link as the first real execution capability.
- PostgreSQL persistence for requests, plans and actions.

## Out Of Scope

- Real Meituan/Dianping booking.
- Real restaurant queue integration.
- Real payment.
- Real food ordering.
- Multi-user voting.
- Full merchant console.
